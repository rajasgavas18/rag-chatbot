# 🤖 RAG Document Q&A Chatbot

An AI-powered chatbot that answers questions from your PDF documents using RAG (Retrieval Augmented Generation).

## 🌐 Live Demo
👉 [rag-chatbot-rajas.streamlit.app](https://rag-chatbot-rajas.streamlit.app)

## ✨ Features
- 📁 Multiple PDF Support
- 💬 Conversational Memory
- 📊 Resume Score & Suggestions
- 🌐 Multilingual (English / Hindi / Marathi)
- 🎨 Beautiful Dark UI

## 🛠️ Tech Stack
- **LangChain** — Document processing
- **Groq LLaMA 3.3-70B** — AI responses
- **Streamlit** — Web interface
- **Python** — Backend

## 🚀 Run Locally

```bash
git clone https://github.com/rajasgavas18/rag-chatbot
cd rag-chatbot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## 📄 Usage
1. Upload any PDF
2. Ask questions in English / Hindi / Marathi
3. Get instant AI answers!
